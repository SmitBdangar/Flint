# Flint - The Local-First AI Developer Toolkit
Flint is the missing layer between raw local AI models (Ollama, LM Studio) and production-ready applications. It provides a robust CLI, Vector Memory (RAG), and a sleek Desktop GUI to help developers build with local AI.

## Features
- **Multi-Backend Auto-Detection:** Automatically spots Ollama, LM Studio, and Llama.cpp servers.
- **In-Place AI Coding:** Use `flint code` to let local LLMs rewrite your files directly.
- **Local RAG (Vector Memory):** `flint memory index` indexes your codebase locally via ChromaDB.
- **ChatGPT-Style Desktop GUI:** A beautiful, responsive Qt interface.
